Drink Management System Client (dmsclient)
==============================

Python3 library with command line interface to interact with the Drink Management System (DMS) of the [Fachschaft TF Uni Freiburg](fachschaft.tf.uni-freiburg.de).

Getting Started
---------------

You can install from git

```bash
git clone git@<git url>:<user>/dmsclient.git
cd dmsclient

python3 setup.py install [--user]

# as development version (cli installed via symlink)
python3 setup.py develop [--user]
```

or simply from PyPi (if available)

```bash
pip install dmsclient [--user]
```
